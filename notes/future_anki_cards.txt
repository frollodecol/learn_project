Как переместиться в другую папку из текущей?
    cd
Как показать содержимое текущей папки?
    dir или ls
Как создать папку через консоль?
    md folderName
Как показать скрытое содержимое директории через консоль виндовса?
    ls -force // ls -a почему-то не работает
Как задать имя пользователя гит?
    git config --global user.name username (не забывай, что config используется без двух дефисов)
Как задать email пользователя в гит?
    git config --global user.email useremail@mail.com
Как посмтреть конфиги гитхаба?
    git config --list
Как "добавить" все файлы для коммиченья в репозитории?
    git add .
Как сделать коммит?
    git commit -m "first commit"
Как залить текущую папку в репозиторий гитхаба?
    git remote add origin git@github.com:frollodecol/learn_project.git (обрати внимание, хоть .git и является "дочерней папкой" репозитория) 
    этой строкой всего лишь произошла инициализация. После этого надо вбить 
    git push -u origin master
Как создать ключ ssh?
    Нужно две команды. Первая - чтоб перейти в нужную папку, вторая - чтоб сгенерировать ключ.
    cd ~/.ssh
    ssh-keygen
    Далее просто жми энтеры. Ключ, который надо отправить на сайт, находится в файле с расширением pub, и он достаточно объёмный. Это не отпечаток, что высвечивается в консоли.
    Будет создано два файла. Приватный ключ (его никому не давай), и публичный (с расширением .pub)
Как клонировать удалённый репозиторий на локальную машину?
    git clone git@github.com:frollodecol/learn_project.git
Как сделать перенос строк в VSCode?
    alt + z
Какой флаг использовать, чтоб добавить пакет в devDependencies?
    npm install --save-dev
Как запустить скрипт через npm?
    Сначала пропиши его в package.json, потом запусти с помощью npm scriptname
Какую команду нужно вбить, чтоб начать использовать npm в проекте?
    npm init
